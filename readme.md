🎮 Tic-Tac-Toe Game (Console-Based)

A simple console-based Tic-Tac-Toe Game developed in C++ as Task 2 of the CodeSoft C++ Programming Internship.

This project allows two players to play Tic-Tac-Toe on a 3×3 grid. Players take turns entering their moves, and the game automatically checks for a winner or a draw. Users also have the option to play multiple rounds without restarting the program.


📌 Project Objective

The objective of this project is to build a two-player Tic-Tac-Toe game while strengthening the understanding of C++ programming concepts such as arrays, functions, loops, conditional statements, and game logic.

✨ Features
 Interactive 3×3 game board
 Two-player gameplay (X & O)
 User-friendly console interface
 Real-time board updates
 Win detection for rows, columns, and diagonals
 Draw detection when all cells are filled
 Automatic player switching
 Play Again option without restarting the application
 Input validation for occupied positions (if implemented)

🎮 Game Rules
The game is played by two players.
Player 1 uses X.
Player 2 uses O.
Players take turns entering the row and column for their move.
The first player to align three identical symbols horizontally, vertically, or diagonally wins.
If all nine cells are filled without a winner, the game ends in a draw.
Players can choose to play another round after the game ends.

🧮 Simple Calculator (C++)

📌 Project Overview


The Simple Calculator is a console-based C++ application that performs basic arithmetic operations. It allows users to enter two numbers, choose an arithmetic operation, and displays the calculated result.


This project was developed as Task 3 of the CodeSoft C++ Programming Internship.


✨ Features

➕ Addition

➖ Subtraction

✖️ Multiplication

➗ Division

📥 User-friendly input and output

⚠️ Handles division by zero

💻 Console-based interface


🛠️ Technologies Used

Programming Language: C++

Compiler: g++ / MinGW / Visual Studio Code

Platform: Windows/Linux/macOS


📖 Concepts Used


Variables

Data Types

User Input and Output

Arithmetic Operators

Conditional Statements (if-else or switch)
  
Basic Error Handling

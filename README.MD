 Number Guessing Game

📌 Project Overview
The **Number Guessing Game** is a simple C++ console-based game where the computer generates a random number, and the user has to guess the correct number. The program provides hints after each guess, telling the user whether the guessed number is **too high** or **too low** until the correct number is found.

This project is developed as part of the **C++ Programming Internship Task**.



 🚀 Features

- Generates a random number automatically.
- Allows the user to enter guesses.
- Provides feedback:
  - 📉 Guess is too low.
  - 📈 Guess is too high.
  - ✅ Correct guess.
- Counts the number of attempts taken by the user.
- Simple and beginner-friendly console interface.

---

 How to Play

1. Run the program.
2. The computer will generate a random number.
3. Enter your guess.
4. The program will give hints:
   - If your guess is smaller → "Too Low!"
   - If your guess is larger → "Too High!"
   - If your guess matches → "Congratulations!"
5. Continue guessing until you find the correct number.
